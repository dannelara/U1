CSS

CSS är ett språk som skapar stilmall till en webbsida. CSS skapar presentationsstilen för hemsidans alla element. Med hjälp av CSS kan vi fixa typsnitt, textstorlek och färg.

Regel i CSS

CSS-regler består av två delar. Den första delen kallas för selektorer. Vi använder selektorer för att bestämma vad som påverkas av CSS-regeln. Du kan hitta mer information gällande selektorer på följande sida: Läs om selektorer. Den andra delen är de egenskaper och värden selektorer har. Vad som är viktigt att tänka på när man gör selektorer är att egenskaperna måste ligga inom ett '{}'-block.

Kaskad i CSS

CSS betyder 'Cascading Style Sheets'. Vilken funktion Style sheet har är givet. Men vad betyder 'Cascading'? När man arbetar med en html-sida finns det flera stilmallar som går att tillämpa. Olika webbläsare innehåller egna stilmallar, dessa är standardmallar och kan ändras genom att vi inför våra egna stilmallar. Användare kan ha egna stillmalar kopplade till deras webbläsare. Sedan har vi de stilmallar skaparen av html-sidan har lagt till. Det är här Kaskaden tar över. Kaskaden beskriver vilken prioritet de olika stilmallar har dvs vilken stilmall som ska visas. Följande är hur kaskade prioriterar stilmallar:

Webbläsarens inbyggda regler -> Användarens egna regler -> Upphovsmannens regler -> Användarens regler som är utmärkta med (Important) -> Upphosmannens regler som är utmärkta med (Important)

Om olika regler har liknade funktioner så har alltså den femte regeln företräde. Detsamma gäller regeln fyra och regeln tre. Vi kan säga att kaskaden rinner neråt som i en hierarki.
